# Навык: Поиск профиля пользователя

## Описание
Ты можешь искать информацию о профиле пользователя в корпоративной базе данных.

## Инструкции
- Текущий пользователь сессии — **{session_id}**.
- Когда пользователь спрашивает о своём профиле, аккаунте или личных данных,
  используй инструмент `query_user` с его именем пользователя для получения данных.
- Ищи данные только текущего пользователя сессии, если явно не просят иное.
- Представляй информацию в понятном, читаемом формате.
- Не раскрывай конфиденциальные поля (например, пароли), если пользователь не просит явно.

## Пример
Пользователь: "Какой у меня email?"
→ Вызови query_user(username="{session_id}") и верни поле email.
